# 🛒 **Multi-Vendor E-Commerce Platform**

## 🚀 **Full Stack MERN Marketplace**

A **full-stack Multi-Vendor E-Commerce platform** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** that allows **multiple vendors to register, manage products, and sell through a centralized marketplace**.

The platform includes a **complete vendor management system**, **admin controls**, and a **modern customer shopping experience**.

Vendors can manage **products, orders, and inventory through a dashboard**, while users can **browse products, add items to cart, and place orders securely**.

The goal of this project was to build a **scalable marketplace architecture** while learning **role-based systems, backend API design, and full-stack development**.

# 📊 **System Architecture**

The platform follows a **multi-role architecture**:

**User → Browse Products → Cart → Checkout**

**Vendor → Manage Products → Manage Orders → Dashboard**

**Admin → Manage Vendors → Monitor Platform**

This structure ensures **scalability and secure role-based access control**.
